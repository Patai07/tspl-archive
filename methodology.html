กระบวนการวิเคราะห์ที่เป็นระบบ เพื่อความถูกต้องทางวิชาการและจริยธรรมในการนำมรดกวัฒนธรรมไปใช้งานออกแบบ

3-Layer Semiotic Analysis

Visual Reading Framework

Layer 01
ลักษณะทางกายภาพ (Physical)

วิเคราะห์รูปทรงเรขาคณิต เส้นสาย คู่สีดั้งเดิม และจำแนกหน่วยคำทัศน์ (Visual Morphemes)

Layer 02
วากยสัมพันธ์ (Syntactic)

วิเคราะห์ระบบการจัดวาง (Syntax) ความสมมาตร แกนหลัก และจังหวะ

Layer 03
นัยสัญลักษณ์ (Semantic)

วิเคราะห์นัยสัญลักษณ์ (Connotation) บริบททางวัฒนธรรม และความอ่อนไหว

Usage Protocol

Design Ethics & Protocol

ข้อกำหนดเชิงจริยธรรมสำหรับนักออกแบบในการนำคลังลวดลายดิจิทัลไปต่อยอด รักษาความสมบูรณ์ของต้นฉบับและไม่บิดเบือนความหมายเชิงสัญลักษณ์

ต้องคงไว้เสมอ
  • แกนสมมาตรหลักของลวดลายMain symmetry axis
  • จุดศูนย์กลางและดอกกลาง (Rosette)Center point and rosette
  • สัดส่วนระยะห่างของก้านเถาVine branch proportions
  • การอ้างอิงแหล่งที่มาทุกชิ้นงานSource attribution always
ปรับได้ในขอบเขต
  • น้ำหนักเส้นและการเหลื่อมเงาLine weight and shading
  • ระบบ Layer การซ้อนทับลวดลายLayer stacking system
  • สีภายในพื้นที่ปิดของลวดลายFill colors of closed paths
  • สเกลและขนาดนำไปใช้งานApplication scale / size
ข้อห้ามเด็ดขาด
  • ตัดทอนหรือเพิ่มหน่วยคำทัศน์ใหม่Alter or add visual morpheme units
  • สร้างลายใหม่โดยอ้างว่าดั้งเดิมClaim invented patterns as original
  • อ้างอิงยุคสมัยที่ยังไม่ผ่านการยืนยันClaim unverified historical era
  • ใช้ลายศักดิ์สิทธิ์ในบริบทเชิงพาณิชย์Commercialize sacred patterns
