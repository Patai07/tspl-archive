Capture
Protocol

กระบวนการบันทึกหลักฐานภาคสนามโดยละเอียด เพื่อรักษาข้อมูลบริบทและพิกัดที่แม่นยำที่สุดก่อนเข้าสู่กระบวนการดิจิทัล Detailed field documentation process to preserve context and precise coordinates before digitization.

System Module Capture_v2.0
Coordinate Mode WGS-84 / UTM
Ref Doc. #01/Capture

การถ่ายภาพละเอียด Ultra High-Res

การถ่ายภาพความละเอียดสูงในมุมมอง Orthographic ปรับแก้ระนาบให้ตรงแนวระเบียบวิธี เพื่อลดความเพี้ยนของสถาปนิกรูปทรง High-precision photography in orthographic view to eliminate proportion distortion for architectural accuracy.

บันทึกพิกัดและบริบท Context Mapping

ระบุพิกัดสากล (WGS84) และตำแหน่งของลวดลายภายในโครงสร้างอาคาร เพื่อจัดลำดับความสำคัญของชุดข้อมูล Logging global coordinates (WGS84) and specific architectural placements to establish data hierarchy and context.

การวิเคราะห์เทคนิค Process Analysis

วิเคราะห์ประเภทวัสดุและเทคนิคช่างโบราณ ทั้งงานสลัก งานปั้น และงานเขียน เพื่อให้ได้รูปทรงที่ตรงตามจิตวิญญาณดั้งเดิม Analyzing traditional craftsmanship techniques—stucco, carving, and Painting—to match original artistic intentions.
