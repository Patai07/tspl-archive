01. เก็บข้อมูล
ขั้นที่ 01 • การบันทึกหลักฐาน

Capture
Protocol

กระบวนการบันทึกหลักฐานภาคสนามโดยละเอียด เพื่อรักษาข้อมูลบริบทและพิกัดที่แม่นยำที่สุดก่อนเข้าสู่กระบวนการดิจิทัล

System Module Capture_v2.0
Coordinate Mode WGS-84 / UTM
Ref Doc. #01/Capture

การถ่ายภาพละเอียด

การถ่ายภาพความละเอียดสูงในมุมมอง Orthographic ปรับแก้ระนาบให้ตรงแนวระเบียบวิธี เพื่อลดความเพี้ยนของสถาปนิกรูปทรง

บันทึกพิกัดและบริบท

ระบุพิกัดสากล (WGS84) และตำแหน่งของลวดลายภายในโครงสร้างอาคาร เพื่อจัดลำดับความสำคัญของชุดข้อมูล

การวิเคราะห์เทคนิค

วิเคราะห์ประเภทวัสดุและเทคนิคช่างโบราณ ทั้งงานสลัก งานปั้น และงานเขียน เพื่อให้ได้รูปทรงที่ตรงตามจิตวิญญาณดั้งเดิม
